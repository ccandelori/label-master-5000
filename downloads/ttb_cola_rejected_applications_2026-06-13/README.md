# TTB COLA Rejected Applications Fetch Attempt

No rejected, denied, or disapproved application PDFs were found through the public TTB COLA Registry.

TTB's public registry page says the public database provides access to approved, expired, surrendered, or revoked COLAs. The public advanced search page repeats the same status set. A live scan of recent public registry records found statuses such as APPROVED, EXPIRED, and SURRENDERED, but no REJECTED, DENIED, or DISAPPROVED records.

The public registry does expose printable application pages for public COLA records, but rejected/denied applications appear to be outside the public registry surface.
